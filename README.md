# 🎧 Overloadr - Extensão Amazon Afiliados

Extensão para Chrome/Edge que adiciona automaticamente o código de afiliado do podcast Overloadr quando você navega em produtos na Amazon Brasil.

## 📋 O que faz

- ✅ **Automático**: Adiciona `tag=overloadr-20` em produtos da Amazon Brasil
- ✅ **Transparente**: Funciona silenciosamente no navegador
- ✅ **Seguro**: Não coleta dados pessoais
- ✅ **Simples**: Apoie o podcast sem esforço extra

## 🚀 Como usar

### Instalação (Recomendado)
1. Baixe o arquivo ZIP na seção [Releases](../../releases)
2. Extraia o arquivo ZIP para uma pasta
3. Abra Chrome/Edge e vá em `chrome://extensions/` ou `edge://extensions/`
4. Ative o "Modo do desenvolvedor"
5. Clique em "Carregar sem compactação"
6. Selecione a pasta extraída

### Instalação alternativa (Código fonte)
1. Clone ou baixe este repositório como ZIP
2. Extraia para uma pasta
3. Siga os passos 3-6 acima

## 🛡️ Privacidade

- Funciona apenas em páginas da Amazon Brasil
- Não rastreia nem coleta dados do usuário
- Código fonte aberto e auditável

## 💡 Como funciona

Quando você visita um produto na Amazon Brasil, a extensão automaticamente adiciona o código de afiliado do Overloadr na URL, garantindo que o podcast receba crédito pela compra.

**Exemplo:**
- Antes: `https://amazon.com.br/dp/B08N5WRWNW`
- Depois: `https://amazon.com.br/dp/B08N5WRWNW?tag=overloadr-20`

## 🤝 Apoie o Overloadr

Esta extensão é uma forma simples de apoiar o podcast Overloadr. Use suas compras normais da Amazon para ajudar o conteúdo que você gosta!
